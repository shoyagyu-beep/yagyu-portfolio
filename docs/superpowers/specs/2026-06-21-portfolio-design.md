# ヤギュウショウ ポートフォリオサイト — レイアウト設計書
2026-06-21

---

## 1. 全体構造

### レイアウト原則

**左右分割型（テキストインデックス＋ホバープレビュー）**  
参照: rikako-nagashima.com

| 領域 | 幅 | 内容 |
|---|---|---|
| 左柱 | 約30% | ロゴ / ナビ / ページ固有のリスト |
| 右エリア | 約70% | 写真プレビュー / 本文 / 空白 |

- 左柱の背景は**透過**（白背景なし）。ページ背景色がそのまま見える
- 左右の区切り線: 細い `border-right` または なし（実装時に要確認）
- 左柱は全ページ共通の骨格として維持する

---

## 2. ページ別レイアウト

### Home

| 領域 | 内容 |
|---|---|
| 左柱 | ロゴ → ナビ → 短い言葉（テキスト） → 最新Noteへのリンク |
| 右エリア | **空白**（何も表示しない） |

- 右側の余白そのものが表現となる
- EN/JA 切り替えなし

---

### Projects

| 状態 | 内容 |
|---|---|
| デフォルト | 左: 作品タイトル一覧 / 右: 空白 |
| ホバー中 | 右の画像が**画面左端まで全幅**に広がり、左柱のテキストの後ろに滲む |
| クリック | Project Detail へ遷移 |

- 左柱のテキスト (`z-index` 上位) が画像に重なって読める状態を維持する
- EN/JA 切り替えなし（一覧ページ）

---

### Project Detail（パターンA / B / C）

| 領域 | 内容 |
|---|---|
| 左柱 | ロゴ → ナビ → 作品リスト（現在の作品をハイライト） → 前後ナビ（← prev · next →） |
| 右エリア | 作品コンテンツ（パターンA/B/Cによる） |

- 右エリアの右上隅に **EN** ボタン（EN版がある場合のみ表示）
- パターンA: ステートメント → 写真 → 制作記録 → 関連Note → 展示歴 → クレジット
- パターンB: 写真 → ステートメント(短文) → 以下同
- パターンC: 現在の状況 → 対話・制作記録(時系列) → 写真 → 問い → 共同制作者

---

### Notes

| 領域 | 内容 |
|---|---|
| 左柱 | ロゴ → ナビ → Note一覧（日付＋種別: Poem / フィールドノート / Daily など） |
| 右エリア | 選択中のNoteの本文 |

- Notesは英語版なし。EN表示時も日本語のまま
- EN/JA 切り替えなし

---

### About

| 領域 | 内容 |
|---|---|
| 左柱 | ロゴ → ナビ → セクションリンク（Biography / Statement / ＋展示歴 / ＋出版歴 / Contact） |
| 右エリア | 本文。展示歴・出版歴は「＋」で開閉するアコーディオン |

- 右エリアの右上隅に **EN** ボタン
- 商業活動への導線は末尾に短く記載

---

## 3. モバイル

左柱を解体し、縦1カラムに変換する。

| 要素 | モバイルでの扱い |
|---|---|
| ロゴ | ページ上部（常時表示） |
| ナビ | **フッター固定**（works / notes / about） |
| コンテンツ | 全幅で縦スクロール |

- Projects ホバープレビューはタップで展開（またはサムネイル表示に切り替え）
- ブレークポイントは実装時に決定（目安: 768px 以下で1カラム）

---

## 4. フォント

| 用途 | フォント |
|---|---|
| 欧文（ナビ・見出し・英語本文） | **EB Garamond**（Google Fonts） |
| 和文（日本語本文・作品名） | **Noto Serif JP**（Google Fonts） |

- ウェイト: 基本 400、強調 500
- 見出しと本文のコントラストは控えめに

---

## 5. ロゴ

- ファイル: `logo/logo1.svg`
- viewBox: `0 0 360.31 55.03`（横長）
- 配置: 左柱最上部、`width: 100%` で列幅にフィットさせる
- 色: SVG内のまま（`#231815`、ほぼ黒）
- ダークモード対応が必要な場合は別途白抜きバージョン（logo1-white.svg）を用意する

---

## 6. EN / JA 切り替え

- 表示箇所: **右エリアの右上隅**、小さなテキストリンク
- 表示対象ページ: Project Detail（EN版あり）、About のみ
- Notes・Home・Projects一覧ページには表示しない
- Astro 組み込み i18n で実装（`/en/` プレフィックス）
- 英語版のないページは日本語にフォールバック

---

## 7. アニメーション・インタラクション

| インタラクション | 挙動 |
|---|---|
| Projectsホバー | 右エリアに写真がフェードイン（`opacity` トランジション、約200ms） |
| ページ遷移 | フェードのみ（派手なエフェクトなし） |
| About 展示歴・出版歴 | 「＋」クリックでアコーディオン展開（高さアニメーション） |
| その他 | 装飾的なホバーエフェクトなし |

---

## 8. 配色

- ベース: 白またはオフホワイト（`#ffffff` / `#fafafa`）＋ 黒（`#1a1a1a`）
- アクセントカラー: 使わない
- 写真が画面に滲む際の背景色は写真そのものの色に依存（グラデーション等で調整しない）

---
